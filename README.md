# 🎟️ RaigaStore

Landing Page modern untuk penjualan aplikasi premium seperti Netflix, Spotify, CapCut Pro, YouTube Premium, Disney+, Discord Nitro, Canva Pro, dan lainnya.

Website ini dibuat menggunakan **HTML, CSS, dan JavaScript Vanilla** tanpa framework sehingga ringan, cepat, dan mudah dikembangkan.

- 🎨 Modern Ticket Style UI
